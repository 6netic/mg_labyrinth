This application is a Labyrinth game made with Pygame in Python language.
To install the requirements, just type pip install -r requirements.txt (this will install pygame library).
To launch the game press F1.
The character is MacGyver and you must guide him to the guardian by using up, down, left and right keys from the keyboard.
Before reaching the guardian MacGyver must pick up the three elements on the labyrinth otherwise game will be lost.
In this version there's only one level of difficulty.
